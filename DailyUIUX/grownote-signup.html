GrowNote — Signup Flow

The project started from a simple prompt: create a sign up experience for a digital product.

The brief was intentionally open, allowing flexibility to define the product, audience, and overall experience. I chose to design a mobile onboarding flow for a plant care app, focusing on simplicity and clarity.

Problem

Users often abandon signup flows due to complexity, too many inputs, or unclear steps. This creates frustration and prevents users from completing the onboarding process.

Many signup experiences also lack visual guidance, making the process feel repetitive and less engaging.

Process

I started by exploring low-fidelity wireframes to define the structure of the onboarding flow and identify the essential inputs required.

The focus was on reducing cognitive load by simplifying the number of steps and organising content in a clear and predictable way. From there, I refined the layout and visual hierarchy before moving into high-fidelity designs.

Visual Identity

Design

The final design uses a soft, nature-inspired colour palette to reflect the theme of plant care and create a calm, welcoming experience.

A clean layout and rounded UI elements help make the interface feel approachable, while clear input fields and spacing guide users through the process step by step.

Key features include:

  • Minimal onboarding flow with only essential inputs
  • Clear and accessible form fields with visual feedback
  • A strong visual hierarchy to guide user attention
  • A friendly and approachable interface aligned with the app's theme

Result

The final concept simplifies the signup experience by reducing friction and making each step easy to understand.

It creates a more engaging onboarding flow while maintaining clarity, helping users complete the process quickly and confidently.

This project helped me explore how visual design and structure can improve usability in form-based interactions.
